# wallet-team5

Репозиторий для сдач домашних работ на курсе SkyPro "Практика реальной разработки (Мемо)":

Спринт 1. Старт проекта и базовая верстка. 
Задачи:

1. Инициализация проекта, настройка окружения.
2. Создание единого Header.
3. Верстка формы на главной странице.
4. Верстка страниц регистрации и авторизации.
5. Верстка диаграммы на странице Анализов (без логики).
6. Верстка таблицы на главной странице.
7. Верстка календаря на странице Анализов (без логики). Выполнена верстка календаря на странице Анализа расходов согласно макету. <--

Спринт 2. API и логика.
Задачи:

1. Создание базовой архитектуры, настройка роутинга.
2. Логика входа, переходы между страницами, сохранение токена.
3. Создание файла API для авторизации/регистрации.
4. Валидация полей, отображение ошибок при регистрации/авторизации.
5. Реализация логики выхода из приложения.
6. Динамическая диаграмма (данные → визуализация).
7. Добавлена логика к календарю - возможность выбора периода на десктопе без кнопки и с кнопкой в мобильной версии согласно макету. <--

1. Адаптивность: регистрация, авторизация.
2. Реализация удаления транзакций.
3. Реализация логики выхода из приложения.
4. Адаптивность: главная страница
5. Рефакторинг, чистка кода (для всех).
6. Адаптивность: страница аналитики - Диаграммы.
7. Адаптивность: страница аналитики - Календарь.
